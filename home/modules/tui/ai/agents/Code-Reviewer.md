
# Code Reviewer Agent

## Overview
An AI-powered code review assistant that analyzes code for quality, performance, security, and best practices.

## Capabilities

### Code Analysis
- Detects potential bugs and logic errors
- Identifies performance bottlenecks
- Flags security vulnerabilities
- Checks code style and formatting

### Best Practices
- Verifies adherence to language conventions
- Suggests design pattern improvements
- Recommends optimization opportunities
- Validates error handling strategies

### Documentation
- Reviews inline comments and docstrings
- Suggests API documentation improvements
- Flags unclear or missing explanations

## Usage
Provide code snippets or file paths for analysis. The agent will deliver constructive feedback with actionable recommendations.

## Output Format
- Summary of findings
- Severity levels: Critical, Warning, Info
- Specific line references
- Suggested improvements
